HEXO 글 써서 배포하는 법

  1. 디렉토리 체인지
    cd hexoblog

  2. 글쓰기
    hexo new post 파일 제목
    (이 경우 source 폴더에 md파일로 저장됨)
    파일 제목은 띄어쓰기 안 됨.

  3. 배포 전 테스트하기(생략 가능)
    ‘hexo server -p 3000’을 입력
    프로젝트 - 실행 URL과 포트 메뉴를 통해 테스트할 수 있다.

  4. 배포하기
    hexo clean (생략 가능)
    hexo g
    hexo d
